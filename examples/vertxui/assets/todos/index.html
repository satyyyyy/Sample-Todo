VertxUI is 100% Java 100% async client & server framework, which abstracts Javascript and HTML behind pure Java in the browser, and has good support for the asynchronous lightweight server VertX. You can share the same data classes serverside and clientside, and you can use Fluent HTML to describe pure model-to-view methods. VertxUI is very thin, making front-end web development a piece of cake if strong-typed Java is your language.
